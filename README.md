# UAC Predictive Intelligence System

## Overview
This project predicts future demand in the U.S. Department of Health and Human Services (HHS) Unaccompanied Alien Children (UAC) Program.

## Objectives
- Forecast Children in HHS Care
- Predict Discharge Demand
- Identify Capacity Stress
- Provide Early Warning Indicators

## Dataset
HHS Unaccompanied Alien Children Program Daily Data

## Technologies
- Python
- Pandas
- Scikit-Learn
- Streamlit
